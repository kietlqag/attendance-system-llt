
  # Attendance Management System

  This is a code bundle for Attendance Management System. The original project is available at https://www.figma.com/design/shoJhOzq8NmpdrBqK7iiAd/Attendance-Management-System.

  ## Running the code

  Run `npm i` to install the dependencies.

  Create `.env` from `.env.example` and fill in your Firebase project values.

  Run `npm run dev` to start the development server.
